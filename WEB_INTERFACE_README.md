## ğŸš€ Usage

### Basic Usage
Simply run the automation pipeline as usual:

```bash
python app.py
```

After completing Steps 1-6 (data scraping, ML training, predictions), the web interface will automatically launch at **http://localhost:5000** and open in your default browser.

### Command-Line Options

```bash
# Run with custom port
python app.py --port 8080

# Skip web interface (for automation/headless mode)
python app.py --no-web

# Force rebuild datasets
python app.py --force-rebuild

# Force retrain model
python app.py --retrain-model

# Combine options
python app.py --force-rebuild --retrain-model --port 8080
```

### Stopping the Server
Press **CTRL+C** in the terminal to stop the web server.

## ğŸ› Troubleshooting

### Port Already in Use
If port 5000 is already in use:
```bash
python app.py --port 8080
```

### Web Interface Not Opening
- Check that Flask is installed: `pip install flask plotly`
- Try manually opening: http://localhost:5000 in your browser
- Check logs for error messages

### No Predictions Showing
- Ensure you've run the full pipeline first (Steps 1-6)
- Check that `fight_predictions_summary.csv` exists in `ufc_data/`
- Verify data files are not corrupted

### Charts Not Rendering
- Ensure internet connection (for CDN resources)
- Check browser console for JavaScript errors
- Try refreshing the page
